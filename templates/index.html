Welcome to PEHCHANO

Identify Urdu text instantly from any Perso-Arabic script input using our intelligent language detection tool

About PEHCHANO

PEHCHANO is a web-based tool designed to detect whether text written in the Perso-Arabic script is Urdu or not. Powered by a custom-trained machine learning model, it supports real-time detection with a simple, user-friendly interface

How It Works

Users can type or paste text in the input box using any Perso-Arabic script (e.g., Urdu, Arabic, Persian). The system analyzes the input using a trained machine learning model and instantly tells whether the text is Urdu or not, along with a confidence score.

Features
  • Real-time language identification
  • Confidence score for transparency
  • On-screen Urdu keyboard for easy typing
  • Guest access with 10 free identifications
  • Unlimited use for logged-in users
Why PEHCHANO?

Unlike generic language detectors, PEHCHANO is purpose-built to focus solely on identifying Urdu from closely related Perso-Arabic languages. It’s ideal for linguists, researchers, and developers working with multilingual datasets in Urdu.
